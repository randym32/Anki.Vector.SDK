    Class NavMapGridNode

    A node in the NavMap

    Leaf nodes contain content, all other nodes are split into 4 equally sized children.

    Child node indices are stored in the following X,Y orientation:

    +---+----+---+ | ^ | 2 | 0 | +---+----+---+ | Y | 3 | 1 | +---+----+---+ | | X->| | +---+----+---+
